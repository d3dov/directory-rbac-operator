# Roadmap

- Add keyless Cosign signing and verification guidance for release artifacts.
- Consider configurable, secure referral handling after defining trust and
  credential boundaries.
- Add optional directory implementations behind the `Grouper` interface when
  they have a clear, supportable membership contract.
- Continue expanding compatibility validation for supported directory servers.

The project intentionally does not plan to become an authentication service,
directory server, or UI.
